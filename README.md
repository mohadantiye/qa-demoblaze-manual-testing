# qa-demoblaze-manual-testing

project overview
This repo contains manual testing of the demoblaze website

The scope will be;
login functionality
signup functionality
product browsing 
cart system 
checkout process
Contact Form Validation

## Testing Types
- Functional Testing
- Negative Testing
- Exploratory Testing

## Key Findings
- Users can place orders without logging in (High severity)
- Weak input validation in signup, checkout, and contact form
- Session persists after closing browser
